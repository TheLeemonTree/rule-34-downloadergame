# rule-34-downloadergame

## Description


This is a downloader for websites that uses booru, such as gelbooru, r34 and so on, it currently only works on r34 and has manny bugs.

This is made in godot c#

## How to use it

Go to settings, put your api key, then put a path to a download folder.

Save, then go to the main page and put a tag to download.

NOTE THE BLACKLIST SYSTEM DOES NOT WORK, I MEAN TECHNIQLY IT DOES, BUT IT DOES NOT SAVE


## Goals

The goal of this project is to make a very good booru downloader

- [ ] Check duplicates on disk to skip download

- [ ] Make blacklist to be saved on disk

- [ ] Other booru websites

- [ ] Possible e621 as well

- [ ] Download only one type of post (mp4, gif, jpeg, so on)

- [ ] Conveyour system, so you can prompt to download in sequence

- [ ] Pause download

- [ ] Cancel download


## How to run

There is not .exe at the moment, but you can download the repo and run it locally on your system, you will need godot .net version installed, latest version, also c# and .net needs to bei nstalled as well.
